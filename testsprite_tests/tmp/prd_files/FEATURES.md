# ServisLog+ Features

## Overview
ServisLog+ provides comprehensive workshop management features.

## Transaction Management
- Create service transactions
- Status tracking: Antri -> Dikerjakan -> Selesai -> Lunas
- Complaint and mechanic notes
- Odometer recording
- Service recommendations

## Customer Management
- Customer database (name, phone, address)
- Vehicle management per customer
- Service history tracking
- Customer notes

## Inventory (Katalog)
- Item catalog (spareparts, oils, accessories)
- Service master list
- Stock monitoring with minimum threshold
- Barcode scanning support

## Staff Management
- Technician and admin data
- Role-based access
- Commission tracking

## Statistics
- Revenue reports
- Service analysis
- Product performance
- Technician evaluation

## Security Features
- PIN authentication
- Biometric (fingerprint/face)
- Data encryption (AES-GCM)
- Remote wipe capability

## Backup & Sync
- Local backup
- Google Drive integration
- Cloud sync with Firestore
