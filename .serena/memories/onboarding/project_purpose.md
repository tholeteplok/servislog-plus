# Project Purpose
The ServisLog Core project is a Flutter application designed for managing service transactions and customer data. It aims to provide a premium, modern, and efficient user experience for service-based businesses, initially focusing on motorcycle diagnostics and general service logging.

# Key Goals
- Modern UI with glassmorphism and centralized theme management.
- Efficient customer (Pelanggan) and transaction management.
- Local data persistence using ObjectBox.
- Future integration with Firebase for cloud sync.
