# Tech Stack
- Frontend: Flutter (Dart).
- UI Framework: Flutter Material with premium custom themes.
- State Management: Riverpod (v2 with Notifiers).
- Database: ObjectBox (local).
- Theming/Styling: Google Fonts (Inter), SolarIcons, and Glassmorphism effects with `GlassCard`.
- Navigation: Custom `MainScreen` with `FloatingNavBar` using `PageView`.
- Misc: `uuid`, `intl`, `logger`, `shared_preferences`, `image_picker`, `path_provider`.
- Planned: Firebase Core/Auth/Firestore for future sync features.

# Commands
- `flutter pub get`: Install dependencies.
- `flutter run`: Run the project.
- `dart run build_runner build --delete-conflicting-outputs`: Generate code for ObjectBox and other generators.
- `flutter analyze`: Perform static code analysis.
- `flutter test`: Run automated tests.
- `dart format .`: Format the code.
